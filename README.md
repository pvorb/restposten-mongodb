Restposten-MongoDB
==================

MongoDB engine intended to be used with
[restposten](https://github.com/n-fuse/restposten.git).


Installation
------------

    npm install restposten-mongodb


API Documentation
-----------------

You can find the API docs at <http://n-fuse.github.com/restposten-mongodb/>.
